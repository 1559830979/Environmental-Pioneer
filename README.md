## 项目简介
现在社会越来越加重视垃圾回收和利用，但人们对于垃圾的分类回收等知识了解程度不高，导致了我们的垃圾有效分类程度不高，无法充分有效的利用垃圾资源，我们的项目就是抓住了这个点，希望通过这个项目对大家的垃圾分类知识进行普及，提升国民的对于垃圾分类的认知，从而提升垃圾分类有效性，使得垃圾资源能够更低成本的回收，同时保护环境，创建一个更美好的地球。
